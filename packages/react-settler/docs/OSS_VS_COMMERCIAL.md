# OSS vs Commercial: Strategic Feature Split

This document explains the strategic separation between open-source and commercial features.

## Philosophy

### OSS Tier: "Great Developer Experience, Core Functionality"

**Goal:** Make React.Settler the best open-source reconciliation protocol

**Includes:**

- Everything developers need to build reconciliation UIs
- Excellent developer experience
- Security basics (XSS protection, validation)
- Mobile and accessibility support
- Core components and protocol

**Why:** Drive adoption, build community, establish React.Settler as the standard

### Commercial Tier: "Advanced Integrations & Enterprise Features"

**Goal:** Monetize advanced features that provide clear business value

**Includes:**

- Platform integrations (Shopify, Stripe, MCP)
- Performance optimizations (virtualization)
- Enterprise features (audit logging, telemetry)
- Advanced security features

**Why:** Clear ROI for businesses, sustainable revenue model

- **Goal:** Adoption, community growth

### Commercial: $99/month
